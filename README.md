# Guitar Tuner

A PWA guitar tuner that uses your device microphone for real-time pitch detection.

## Features

- Autocorrelation-based pitch detection with noise rejection
- Auto-detects which string you're playing
- Visual cues: green glow when in tune, directional hints (tune up/down)
- Works offline (PWA)

## Usage

Open [tuner.ponsa.xyz](https://tuner.ponsa.xyz), allow microphone access, and pluck a string.
